## Overview
Stores things like message content, any embeds, etc

```swift
struct Message
```

## Properties

### Public Properties

- `content`: `MessageContent`  
  The main content of the message.
  <Code title="Example" lang="swift" code="Message {
  MessageContent {
  ...
  }
  }"/>

- `embeds`: `[MessageEmbed]`  
  An array of embeds associated with the message.

- `attachments`: `[MessageAttachment]`  
  An array of file attachments for the message.

- `components`: `MessageComponents`  
  The components included with the message.

- `stickers`: `[MessageSticker]`  
  An array of stickers associated with the message.

- `poll`: `MessagePoll?`  
  An optional poll associated with the message.

### Internal Properties

- `_nonce`: `StringOrInt?`  
  A unique identifier for the message to prevent duplication.

- `_content`: `String?`  
  The textual representation of the message content, or nil if empty.

- `_tts`: `Bool?`  
  Indicates whether text-to-speech is enabled for the message.

- `_embeds`: `[Embed]`  
  An array of formatted embeds derived from the message's embeds.

- `_allowed_mentions`: `Payloads.AllowedMentions?`  
  Specifies allowed mentions in the message.

- `_message_reference`: `DiscordChannel.Message.MessageReference?`  
  References another message in Discord.

- `_components`: `[Interaction.ActionRow]?`  
  The interactive action rows converted from the message components.

- `_sticker_ids`: `[String]?`  
  An array of sticker IDs associated with the message.

- `_files`: `[RawFile]?`  
  An array of raw file representations for the attachments.

- `_attachments`: `[Payloads.Attachment]?`  
  An array of attachment payloads, excluding embeds.

- `_flags`: `IntBitField<DiscordChannel.Message.Flag>`  
  A bitfield to represent various message flags.

- `_enforce_nonce`: `Bool?`  
  Indicates whether to enforce the nonce.

- `_poll`: `Payloads.CreatePollRequest?`  
  The poll request payload if a poll exists.

## Public Modifiers

### `tts(enabled) -> Self`
Enables or disables text-to-speech for the message.

- **Parameters**:
  - `enabled`: A `Bool` indicating whether TTS should be enabled (default is `true`).

### `ephemeral(enabled) -> Self`
Sets whether the message is ephemeral.

- **Parameters**:
  - `enabled`: A `Bool` indicating whether the message should be ephemeral (default is `true`).

### `flags(flags) -> Self`
Manually configures message flags, overriding previous modifiers.

- **Parameters**:
  - `flags`: An `IntBitField` representing the desired flags.

### `allowedMentions(kind, roles, users, repliedUser) -> Self`
Sets the allowed mentions for this message.

- **Parameters**:
  - `kind`: An optional array of allowed mention kinds.
  - `roles`: An optional array of role snowflakes to mention.
  - `users`: An optional array of user snowflakes to mention.
  - `repliedUser`: An optional `Bool` indicating whether to mention the user being replied to (default is `true`).
